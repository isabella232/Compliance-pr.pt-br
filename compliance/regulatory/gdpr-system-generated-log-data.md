<span data-ttu-id="41600-p103">**Workplace Analytics**: A ferramenta de exportação do log de dados fornece os dados de uso para os usuários de sua organização que têm permissão para executar relatórios do Workplace Analytics. <span data-ttu-id="41600-p104">**Yammer**: O centro de administração do Yammer permite exportar dados de atividade da conta do usuário.
